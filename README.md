## Quick Start

#### 1. Install pip requirement
``` bash
$ pip install -r requirements.txt
```

#### 2. Run Fast-API Backend
``` bash
$ python tennis_analysis_demo_api.py
```

#### 3. Open WebUI
```` bash
$ google-chrome ./tennis_analysis_demo.html 
````
